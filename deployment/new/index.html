▶ STAGE.001 // SYSTEM UPDATE

Initialize system and prepare for deployment. This stage updates package repositories and upgrades all system components to ensure compatibility.

sudo apt update && sudo apt upgrade -y

▶ STAGE.002 // BOOTSTRAP INIT

Download and execute Stage 1 setup script. This installs essential tools including Git and GitHub CLI for repository access.

wget -qO- https://privitera.github.io/public/stage1-setup.sh | sudo bash

▶ STAGE.003 // AUTH PROTOCOL

Establish secure connection to GitHub. Authentication required for accessing deployment configurations.

gh auth login
PROTOCOL: Select GitHub.com → SSH → Generate new SSH key

▶ STAGE.004 // DEPLOY EXECUTE

Execute Stage 2 deployment with interactive configuration selector. Choose from multiple deployment profiles.

wget https://privitera.github.io/public/deployment/deploy-wrapper.sh && bash deploy-wrapper.sh && rm deploy-wrapper.sh
ACCESS LEVEL: This stage requires private repository access
